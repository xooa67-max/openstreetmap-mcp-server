# 0.1.7 — 2026-05-24

## Fixed

- **`overpass-service.ts`**: Added `User-Agent` header to all Overpass API requests — its absence caused HTTP 406 responses on every Overpass tool ([#4](https://github.com/cyanheads/nominatim-mcp-server/issues/4)).
- **`server-config.ts`**: `NOMINATIM_USER_AGENT` default now reads version dynamically from `package.json` at startup instead of a hardcoded `0.1.0` string ([#5](https://github.com/cyanheads/nominatim-mcp-server/issues/5)).
