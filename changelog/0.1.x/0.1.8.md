# 0.1.8 — 2026-05-24

## Changed

- **`nominatim-format.ts`** (new helper) — `appendPlaceLines` extracted from three copy-pasted blocks in `nominatim-geocode`, `nominatim-lookup`, and `nominatim-reverse`; all three now import the shared helper.
- **`overpass-tag-input.ts`** (new helper) — `resolveTagInput` extracted from duplicate validation logic in `overpass-query-bbox` and `overpass-query-nearby`.
- **`nominatim-lookup.tool.ts`** — OSM ID set built with `flatMap` instead of `map + filter`.
- **`overpass-service.ts`** — `'remark' in data` narrowed to typed property access via `OverpassResponse & { remark?: string }`.
- **`nominatim-service.ts`** — `setIfTruthy` helper replaces 12 individual `if` guards for query-param population; cache hit guards use `!= null` instead of `!== null`.
- **Error codes** — `invalid_input` (`nominatim_geocode`), `invalid_id_format` (`nominatim_lookup`), `invalid_tag` ×2 (`overpass_query_bbox`, `overpass_query_nearby`), `query_error` (`overpass_query_raw`) corrected from `InvalidParams` to `ValidationError`.
- **`mcp-ts-core`** ^0.9.7 → ^0.9.9 (fuzzTool pre-parse fix, ZodArray minLength, fast-check optional peer dep, `@hono/node-server` ^2.0.3 → ^2.0.4).
- **Skills** — `git-wrapup` v1.0 added; `release-and-publish` v2.5 → v2.6.
- **`CLAUDE.md`** — framework version reference updated; "Close the loop on issues" rule added.
